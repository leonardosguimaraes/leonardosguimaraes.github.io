Sobre mim

Meu nome é Leonardo Soares Guimarães moro na cidade do Rio de Janeiro. Sempre trabalhei com vendas, porém como gosto da área de tecnologia resolvi me tornar um desenvolvedor Front-end realizando a minha transição de carreira. Tenho bons conhecimentos utilizando as tecnologias HTML, CSS e a linguagem de programação Javascript. No momento estou me aprofundando no Javascript e iniciando os estudos na tecnologia Node e no framework React. Para o desenvolvimento dos projetos utilizo as ferramentas: Visual Studio Code, Git e GitHub. Continuo sempre me aprimorando, com o objetivo de conseguir o primeiro emprego como desenvolvedor Front-End e no futuro me tornar um Desenvolvedor Full Stack.
